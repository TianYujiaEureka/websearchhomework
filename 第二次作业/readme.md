#第二次作业，图片相似度
使用说明
##图片生成
picgenerator进行图片生成，上水印和改尺寸需要使用一部分的时候注释掉另一部分
###文件名及其含义
resize：改变长和宽（/4）
resize_w:改变宽度（/3）
mark：加水印，水印名为作品作者邦乔彦
##生成hash文件
pic_hash用于生成三种hash
文件路径指向存储图片的文件夹即可自动生成该文件夹内全部图片的三种hash值
**但请保持图片文件夹中只有图片，否则程序会终止**
##比较汉明距离
运行程序即可比较所有图片之间的距离，并将比较结果存入txt文件。名为：Xhash_likemost
